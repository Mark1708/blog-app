# blog-app

## Описание
Это простая реализация API для блога на Rust для практики

## Запуск
> Создайте .env из примера
```shell
sqlx migrate run
cargo run
```

## Полезные ссылки
* API - http://localhost:8080/api/v1
* Swagger - http://localhost:8080/swagger-ui/index.html

## Технологии
- Go 1.77.2
- Cargo 1.77.2
- Actix Web 4.2.1
